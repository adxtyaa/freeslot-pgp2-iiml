How these are measured: counters are stored in abacus.jasoncameron.dev (anonymous increments, no personal data). Unique users counts devices on first-ever visit (clearing browser storage re-counts a device). DAU counts each device once per IST calendar day. Jobs run counts user-triggered free-slot lookups (page open on results, day/time changes, manual refresh), with bursts within 5 seconds collapsed into one. Counting started when analytics was deployed — earlier usage is not included. Larger ranges load in batches because the counter API allows 30 requests per 10 seconds.
